Google Play Store Analysis

1. Project Overview
This project analyzes Google Play Store data to understand app market trends, user behavior, and business insights.

2. Objectives
- Analyze app categories  
- Study ratings and installs  
- Understand pricing trends  
- Perform sentiment analysis  

3. Dataset
- apps.csv -> App details  
- reviews.csv -> User reviews  

4. Tools Used
- Python  
- Pandas  
- NumPy  
- Matplotlib  
- Seaborn  

5. Steps Performed
- Data Cleaning  
- Category Analysis  
- Rating Analysis  
- Install Analysis  
- Price Analysis  
- Sentiment Analysis  

6. Results
- Most apps are free  
- Popular apps have higher ratings  
- Few categories dominate the market  

7. Insights
- Free apps attract more users  
- Ratings impact installs  
- User reviews help improve apps  

8. Recommendations
- Focus on high-demand categories  
- Improve app quality for better ratings  
- Offer free versions to increase reach  

9. Conclusion
Data analytics helps understand app market trends and supports better decision-making.
